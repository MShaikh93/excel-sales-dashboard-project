# 📊 Excel Sales Dashboard Project

## Project Overview

This project demonstrates the creation of an interactive Executive Sales Dashboard in Microsoft Excel using the Superstore Sales Dataset.This project demonstrates the creation of an interactive Executive Sales Dashboard in Microsoft Excel using the Superstore Sales Dataset. The dashboard is designed to help business users monitor sales performance, profitability, customer trends, and regional performance through interactive visualizations and KPI reporting.

The project showcases practical Excel dashboard development techniques commonly used by Data Analysts and Business Analysts for executive reporting.

Business Objective

Design an interactive dashboard that enables users to:

Monitor overall business performance
Analyze sales across product categories
Track profit trends over time
Explore sales performance by state
Identify top-performing customers
Filter results dynamically using slicers
Dashboard Features
Executive KPI Cards
Interactive Pivot Charts
Dynamic Slicers
Category Filtering
Year Filtering
Sales Trend Analysis
Geographic Sales Analysis (Map Chart)
Customer Profit Analysis
Monthly Sales Trend Visualization
Interactive Dashboard Reporting
